# What Do We Mean by *Language*?

A Pluralist Map for the *Language Sciences*.

The paper maps the recurrent sense-clusters that travel under the label *language* across linguistics, cognitive science, anthropology, philosophy, and adjacent fields. For each cluster: paradigm cases, diagnostics, stabilizers, projection target, defeat conditions, slippage risks. HPC theory (Boyd) supplies the methodological backbone: realist standing for a cluster requires projectibility and homeostasis, and slippage between clusters is a diagnosable failure mode.

Closest published precedent: Santana 2016, "What Is Language?" *Ergo* 3.19. The paper engages Santana directly and shows where HPC discipline does work that his three-criteria pluralism does not.
